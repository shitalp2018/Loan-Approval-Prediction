
# Loan Approval Prediction System

Built an end-to-end supervised Machine Learning pipeline using Logistic Regression, KNN, and Naive Bayes to predict loan approval decisions. Performed EDA, feature engineering, binary classification, and model evaluation using Precision, Recall, and F1 Score.

## Overview
Built an end-to-end supervised Machine Learning pipeline to predict whether a loan application should be approved or rejected. The project uses applicant financial, demographic, and credit-related information to automate loan approval decisions and support data-driven lending.

## Objectives
- Predict loan approval outcomes using historical applicant data.
- Compare the performance of multiple classification algorithms.
- Identify important factors influencing loan approval decisions.
- Build a complete Machine Learning workflow from data preprocessing to model evaluation.

## Features
- Loan approval prediction using Machine Learning
- Exploratory Data Analysis (EDA)
- Feature engineering and encoding
- Multiple model comparison (Logistic Regression, KNN, Naive Bayes)
- Performance evaluation using Precision, Recall, and F1 Score
